# Mini Proyecto Python - Operaciones Matemáticas

## Descripción
Script en Python que lee un archivo CSV con operaciones matemáticas,
calcula los resultados y los guarda en el mismo archivo.

## Operaciones soportadas
- SUM: Suma
- SUB: Resta
- MUL: Multiplicación
- DIV: División
- POW: Potencia

## Archivos del proyecto
- mi_proyecto.py: Script principal
- data/math_operations.csv: Archivo con las operaciones y resultados

## Cómo ejecutarlo
1. Instalar pandas: pip install pandas
2. Ejecutar el generador: python mini_proyecto_1.py
3. Ejecutar mi script: python mi_proyecto.py
4. Revisar los resultados en data/math_operations.csv
