# Mini Proyecto Python - Operaciones Matemáticas y Batallas Pokémon

## Proyecto 1: Operaciones Matemáticas

### Descripción
Script en Python que lee un archivo CSV con operaciones matemáticas,
calcula los resultados y los guarda en el mismo archivo.

### Operaciones soportadas
- SUM: Suma
- SUB: Resta
- MUL: Multiplicación
- DIV: División
- POW: Potencia

### Cómo ejecutarlo
1. Instalar pandas: pip install pandas
2. Ejecutar el generador: python mini_proyecto_1.py
3. Ejecutar mi script: python mi_proyecto.py
4. Revisar los resultados en data/math_operations.csv

## Proyecto 2: Batallas entre Pokémons

### Descripción
Sistema de batallas entre Pokémons implementado con programación
orientada a objetos (OOP). Carga los datos desde el archivo pokemon.csv
y permite gestionar y batallar pokémons desde la consola.

### Funcionalidades
- Buscar un Pokémon y ver su tarjeta
- Agregar nuevos Pokémons
- Eliminar Pokémons del sistema
- Modificar el HP de un Pokémon
- Simular batallas entre dos Pokémons

### Cómo ejecutarlo
1. Instalar pandas: pip install pandas
2. Ejecutar: python pokemon_battle.py
3. Seguir el menú interactivo
